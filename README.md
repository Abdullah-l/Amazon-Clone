# Amazon-Clone
## Tools & Versions
- Rails 7.0.1
- Ruby 3.1.0
- SQLite 3.35.0

## Intro

This Task Involves setting up rails mailer to do the following:
- Send a welcome email for every user created
- Send a notification email to all users when a new item is added
